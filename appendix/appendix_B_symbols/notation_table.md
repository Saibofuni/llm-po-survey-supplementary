# Appendix B: Notation Table

符号	含义
"x" 	输入提示
"y" _"w" 	人类偏好的响应（即获胜者）
"y" _"l" 	为非偏好的响应（即失败者）
"y" 	响应序列
"p(" "y" _"w"  "≻" "y" _"l"  "|x)" 	基于Bradley-Terry模型的偏好概率
"q" 	问题
"I" 	图像
"π" _"θ"  "(y|x)" 	"x" 到"y" 的映射，表示在状态"x" 下选择动作"y" 的概率。
"π" _"old"  "(y|x)" 	旧策略，每次更新后，旧策略将被当前策略取代。
"π" _"ref"  "(y|x)" 	参考策略，初始的监督微调模型在整个训练过程中保持固定不变。
"λ" 	调节权重的超参数。
"KL(" "π" _"θ"  "||" "π" _"ref"  ")" 	KL散度，用于衡量当前策略与参考策略之间的差异。
"β" 	KL散度的权重系数。
"σ" 	sigmoid函数
"D" 	包含"(x," "y" _"w"  "," "y" _"l"  ")" 三元组的偏好数据集
"a" 	参考/正确答案
"G" 	采样响应组数
"τ" 	置信度阈值，若模型置信度低于或等于此阈值，则将被选中。
"|y|" 	响应"y" 的总词元数。
"w(x,y)" 	权重
